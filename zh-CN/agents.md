# 智能体与工具

Routero 路由的不只是文本补全。A2A 智能体网关和 MCP 网关将同一套认证、成本追踪和策略强制执行扩展到 AI 智能体和工具服务器。

{: .beta }
> **A2A 和 MCP 目前处于 beta 阶段。** 功能行为和 API 可能会变化。在基于这些端点构建生产工作流之前，请联系你的解决方案工程师。
