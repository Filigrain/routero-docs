# Agents & Tools

Routero routes more than text completions. The A2A Agent Gateway and MCP Gateway extend the same auth, cost tracking, and policy enforcement to AI agents and tool servers.

{: .beta }
> **A2A and MCP are currently in beta.** Feature behaviour and APIs may change. Contact your solutions engineer before building production workflows on these endpoints.
